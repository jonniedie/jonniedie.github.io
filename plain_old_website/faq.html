How much are the fees and how are they collected?

cascaid does not collect any fees itself. There is, however, a flat 2.9% + $0.30 transaction fee from Stripe. Payments are set up as a single transaction from the payer to all the end payees (project maintainers) at once in order to minimize transaction fees.

Okay, so what's the catch?

There isn't one. Much like many of the open source projects cascaid was created to support, the service lives or dies on donations alone. So I guess that's the catch: cascaid might disappear at some point if it can't be supported through donations.

Are recurring donations/subscriptions possible?

Recurring donations aren't set up yet. This is planned, though.

Some of my project's dependencies don't have a cascaid or Stripe account, can I still add them as a downstream receiver?

Unfortunately no. The best thing you can do is tell them about cascaid and see if they'd be willing to join. But please don't spam them about it.

How can I support cascaid?

The best way to support us is to add the cascaid disbursement node to your payment chain. The second best way is to directly donate to the cascaid node.
